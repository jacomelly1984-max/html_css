CSS

CSS (folhas de estilo em cascatas/cascading style sheets) é a linguagem usada para definir a aparencia e o layout das paginas feitas com HTML, controlando cores, fontes, espaçamento, tamanhos e posicionamentos dos elementos; em resumo enquanto o HTML estrutura o conteudo, o CSS deixa tudo visivelmente organizado e bonito.
